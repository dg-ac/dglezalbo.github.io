About IronCalc

IronCalc is a new, modern, work-in-progress spreadsheet engine and set of tools to work with spreadsheets in diverse settings.

Programmed in Rust, you will be able to use it from a variety of programming languages like Python, JavaScript (wasm), nodejs and possibly R, Julia or Go.

The project is in its early stages and I am working on the design of the product. I will keep updating this section with more information as the project progresses.

If you want to know more about the project, you can check the GitHub repository.

Branding and Website

Besides the general UI/UX design of the product, I have been in charge of both branding and website.
